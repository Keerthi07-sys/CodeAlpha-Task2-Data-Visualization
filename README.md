# CodeAlpha Task 2 - Data Visualization

## 📌 Project Overview
This project focuses on transforming raw sales data into meaningful visualizations using Python. Various charts and graphs are created to identify trends, patterns, and insights that support data-driven decision-making.

## 🎯 Objectives
- Transform raw data into visual formats.
- Create clear and informative charts and graphs.
- Identify trends and patterns in sales data.
- Present insights through effective visualizations.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Files Included
- data_visualization.py
- sales_data.csv
- README.md

## 📊 Visualizations Created
- Line Chart (Monthly Sales Trend)
- Bar Chart (Monthly Profit)
- Pie Chart (Sales Distribution)
- Heatmap (Correlation Matrix)

## ▶️ How to Run
1. Install the required libraries:
```
pip install pandas matplotlib seaborn
```

2. Run the Python program:
```
python data_visualization.py
```

## 📈 Output
The project generates multiple visualizations that help understand sales performance, trends, and relationships between different variables.
